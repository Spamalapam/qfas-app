# Q.F.A.S. Protocol Engine

**Quantified Fitness & Accountability System** — personalized to Adam's genome (23andMe v5, 631K SNPs).

## Features

- **📅 Dynamic Schedule** — Hour-by-hour protocol with editable calendar (add/edit/delete custom days)
- **🧬 Genome Panel** — 31 SNPs analyzed with protocol overrides (ACTN3, FTO, COMT, BDNF, APOE, and more)
- **💪 Guided Workout Player** — Step-by-step coaching with form cues, target muscles, rest timer (ACTN3-adjusted 120-150s), weight/rep logging
- **📊 Workout History** — All sessions logged with volume, RPE, and duration tracking
- **🍗 Nutrition** — Genome-adjusted macros (FTO AA → 1.2g/lb protein, 300 cal deficit)
- **💤 Recovery** — Sleep, supplements, and inflammatory stack management

## How to Use

1. Open `index.html` in any browser
2. Navigate tabs: Schedule → Genome → Workouts → History → Nutrition → Recovery
3. Hit **▶ START WORKOUT** on any workout card to launch the guided player
4. Add custom days to your calendar with **+ Add Day**

## GitHub Pages

This is a static site — no build step needed. Enable GitHub Pages on the `main` branch from root.

## Tech

Pure HTML/CSS/JS. No dependencies. Data stored in `localStorage`.
